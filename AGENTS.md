# montage-is-open

This project depends on [OpenMontage](https://github.com/calesthio/OpenMontage) (git submodule at `./OpenMontage/`) for AI-powered video production.

**MANDATORY: Read `OpenMontage/AGENT_GUIDE.md` before responding to ANY video production request.**

The agent should:
1. Work from this directory (`montage-is-open/`)
2. Use OpenMontage's pipeline system via `OpenMontage/` submodule
3. Place generated videos in `projects/` as per OpenMontage conventions
4. For new video requests, follow OpenMontage's pipeline workflow (preflight -> proposal -> script -> scene_plan -> assets -> edit -> compose)

Custom instructions for this project can be added below this line.
